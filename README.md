## `If you want to run locally`

**Run Those command in order.**

- ```sh
   git clone https://github.com/hhhameem/react-tour-de-paris.git
  ```

- ```sh
    cd react-tour-de-paris
  ```
- ```sh
    npm install
  ```
- ```sh
    npm start
  ```
